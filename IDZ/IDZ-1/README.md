# ИДЗ-1 (вариант 6)

Индивидуальное домашнее задание №1 по дисциплине  
«Интеллектуальные информационно-измерительные системы».

## Цель работы

Обработка набора данных средствами языка Python:
генерация временного ряда, вычисление статистических характеристик
и визуализация результатов.

## Постановка задачи

В соответствии с заданием необходимо:

1. Сгенерировать CSV-файл с данными в **5 столбцов** и **10⁶ записей**.
2. Прочитать CSV-файл средствами Python.
3. Найти среднее значение по **второй колонке**.
4. Построить график значений **третьей и четвёртой колонок**
   в зависимости от времени (первая колонка).
5. Для **пятой колонки** построить график медиан по блокам
   объёмом **10⁵ записей**, откладывая по оси X номер блока.

Работа выполнена по **варианту №6**.

## Формирование данных

CSV-файл содержит следующие столбцы:

- `time` — временная шкала (монотонно возрастающая);
- `func1` — случайная величина (базовый уровень + шум);
- `func2` — синусоидальная функция времени с шумовой добавкой;
- `func3` — комбинация `func1` и `func2`;
- `func4` — функция времени с выраженным трендом и шумом.

Параметры генерации функций (period, v1, v2, v3) заданы в коде
в соответствии с вариантом задания.

## Реализованные этапы

Работа разбита на несколько независимых скриптов:

1. **Генерация CSV-файла**
   - формирование файла `test.csv` объёмом 10⁶ строк;
   - запись данных с использованием `csv.DictWriter`.

2. **Чтение данных и вычисление среднего**
   - последовательное чтение CSV-файла;
   - вычисление среднего значения по колонке `func1`.

3. **Построение графиков для колонок 3 и 4**
   - визуализация зависимостей `func2` и `func3` от времени;
   - сохранение графика в формате PNG.

4. **Построение графика медиан**
   - разбиение значений `func4` на 10 блоков по 10⁵ записей;
   - вычисление медианы для каждого блока;
   - построение графика медиан по номеру блока.

## Структура проекта

- `src/` — Python-скрипты:
  - генерация CSV-файла;
  - вычисление среднего значения;
  - построение графиков;
- `data/` — входные и выходные CSV-данные;
- `figures/` — сохранённые графики;
- `report/` — отчёт по работе (обезличенная версия).

В результате выполнения работы:

- вычислено среднее значение по второй колонке;
- построены графики зависимостей для колонок 3 и 4;
- построен график медиан для пятой колонки по блокам.
