# ИДЗ-2

Индивидуальное домашнее задание №2 по дисциплине
«Интеллектуальные информационно-измерительные системы».

## Назначение работы

Целью работы является анализ и обработка экспериментальных данных
с использованием языка Python и стандартных библиотек
для численных расчётов и визуализации.

Работа является самостоятельной и не зависит от результатов ИДЗ-1.

## Содержание проекта

- `src/` — исходный код программных модулей;
- `data/` — входные данные и вспомогательные файлы;
- `figures/` — графики и результаты визуализации;
- `report/` — отчёт по работе (обезличенная версия).

## Общая логика выполнения

1. Загрузка исходных данных.
2. Выполнение требуемых вычислений согласно заданию.
3. Анализ полученных результатов.
4. Визуализация данных и построение графиков.
