# ИДЗ-3

Индивидуальное домашнее задание №3 по дисциплине  
«Интеллектуальные информационно-измерительные системы».

## Цель работы

Реализовать и исследовать алгоритм классификации изображений
на основе многослойного перцептрона (MLP) без использования
готовых нейросетевых фреймворков.

Модель реализована на языке Python с применением библиотеки NumPy,
обучение выполняется методом обратного распространения ошибки.

## Используемая модель

Многослойный перцептрон со следующей архитектурой:

- размер входного слоя: **900** (изображение 30×30, grayscale);
- скрытые слои: **900 → 300**;
- выходной слой: **10 классов**;
- функция активации: **sigmoid**;
- функция потерь: **categorical cross-entropy**;
- регуляризация: **L2**;
- инициализация весов: **Xavier**;
- метод обучения: **batch gradient descent**.

Архитектура сети проверяется при инициализации модели
в соответствии с требованиями задания.

## Структура проекта

- `src/`
  - `idz3.py` — основной сценарий:
    - загрузка датасета и разметки;
    - обучение модели;
    - сохранение и загрузка весов;
    - оценка точности;
    - построение графиков и сохранение примеров распознавания;
    - интерфейс командной строки (CLI).
  - `mlp.py` — реализация класса `MLP`:
    - прямой проход;
    - backpropagation;
    - обучение;
    - сохранение и загрузка весов с проверкой совместимости.
- `data/raw/`
  - датасет изображений и файл `labels.csv`
    (путь указывается через аргументы командной строки).
- `figures/`
  - графики обучения (Loss, Accuracy);
  - примеры распознавания изображений из тестовой выборки.
- `report/`
  - отчёт по работе (обезличенная версия).

Файлы весов модели, логи и промежуточные результаты
в репозиторий не загружаются.

## Формат входных данных

Датасет состоит из:
- изображений дорожных знаков;
- CSV-файла `labels.csv` со следующими полями:
  - `filename` — имя файла изображения;
  - `class` — метка класса.

Для исключения ошибок при повторных запусках
порядок классов формируется **детерминированно**
с использованием стабильной сортировки.
