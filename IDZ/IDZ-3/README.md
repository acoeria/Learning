# ИДЗ-3

Индивидуальное домашнее задание №3 по дисциплине  
«Интеллектуальные информационно-измерительные системы».

## Назначение работы

Целью работы является разработка и исследование алгоритма классификации
данных с использованием методов машинного обучения на языке Python.

В рамках работы реализуется многослойный перцептрон, выполняется
подготовка данных, обучение модели и оценка качества классификации.

## Структура проекта

- `src/` — исходный код:
  - реализация модели многослойного перцептрона;
  - загрузка и предварительная обработка данных;
  - обучение модели и расчёт метрик качества.
- `data/` — входные и подготовленные данные:
  - `raw/` — исходные данные;
  - `processed/` — обработанные выборки (при необходимости).
- `artifacts/` — результаты вычислений:
  - `models/` — сохранённые параметры обученных моделей;
  - `reports/` — численные результаты и метрики;
  - `logs/` — журналы выполнения.
- `figures/` — графики, матрицы ошибок и иллюстрации результатов.
- `report/` — отчёт по работе (обезличенная версия).

## Общая логика выполнения

1. Загрузка исходных данных.
2. Предварительная обработка и подготовка выборки.
3. Обучение модели многослойного перцептрона.
4. Оценка качества классификации.
5. Анализ и визуализация полученных результатов.
