# Learning

Учебный репозиторий с индивидуальными домашними заданиями (ИДЗ)
по дисциплине «Интеллектуальные информационно-измерительные системы».

Репозиторий используется для хранения исходного кода, результатов
вычислений и обезличенных отчётов по учебным проектам.

## Содержание

- `IDZ/IDZ-1` — индивидуальное домашнее задание №1  
  Генерация и обработка табличных данных, вычисление статистических
  характеристик и построение графиков.

- `IDZ/IDZ-2` — индивидуальное домашнее задание №2  
  Формирование и разметка датасета изображений дорожных знаков
  с моделированием искажений (перспектива, фон, шум).

- `IDZ/IDZ-3` — индивидуальное домашнее задание №3  
  Реализация и исследование классификатора на основе
  многослойного перцептрона (MLP).

## Структура проектов

Каждое ИДЗ оформлено как самостоятельный подпроект.
В зависимости от задания используются следующие каталоги:

- `src/` — исходный код;
- `data/` — входные данные и вспомогательные файлы;
- `figures/` — графики, иллюстрации, примеры результатов;
- `report/` — отчёт по работе (обезличенная версия).
