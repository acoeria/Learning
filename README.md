# Learning

Учебный репозиторий с индивидуальными домашними заданиями (ИДЗ)
по дисциплине «Интеллектуальные информационно-измерительные системы».

Репозиторий используется для хранения исходного кода, результатов
вычислений и обезличенных отчётов по учебным проектам.

## Содержание

- `IDZ/IDZ-1` — индивидуальное домашнее задание №1  
  Обработка и анализ данных, генерация CSV-файла, вычисление
  статистических характеристик и построение графиков.

- `IDZ/IDZ-2` — индивидуальное домашнее задание №2  
  Самостоятельный проект по анализу и визуализации данных.

- `IDZ/IDZ-3` — индивидуальное домашнее задание №3  
  Проект с использованием методов машинного обучения
  (многослойный перцептрон, обработка датасета, оценка качества).

## Структура проектов

Каждое ИДЗ оформлено как самостоятельный подпроект и содержит:

- `src/` — исходный код;
- `data/` — входные данные и вспомогательные файлы;
- `figures/` — графики, иллюстрации, блок-схемы;
- `report/` — отчёт по работе (обезличенная версия).

В ИДЗ-3 дополнительно используется каталог `artifacts/`
для хранения результатов вычислений и обученных моделей.
